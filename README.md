# vue_news

Vue CLI3 + Vue Router + Vuex + Axios + Cube UI

一个超级超级简单的新闻类wabAPP，接口数据来自[聚合数据-新闻头条](https://www.juhe.cn/docs/api/id/235),接口可操作度太低，无法订制上拉加载、下拉刷新及Vuex跳转详情页等效果，但相应的代码都已经写好注释了，有兴趣的同学可以自己找or写一个更符合生产环境的接口进行替换
